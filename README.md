A Rolling Start is a 3D open-level game I built in Godot to help children master joystick controls in preparation for using a powered wheelchair. The core of this research project is to determine if a gamified experience is more efficient and enjoyable for learning than traditional teaching methods. Because this was built for research, it is more than just a game; it is a data-collection tool. The system captures movement data, collision frequency, and item collection metrics, all of which can be exported for further analysis. This data allows researchers to track a child’s improvement over multiple play sessions.

The levels are designed to encourage exploration. Each map features unique terrain and hidden items, such as top hats, crowns, and wizard hats. Players can equip any items they find, allowing them to customize their character as they progress. Currently, the game features two distinct environments:
The City: A bustling urban map with large buildings and moving traffic.
The Forest: A natural landscape filled with tall mountains and underground mysteries.

I developed A Rolling Start as a research project through the University of Alabama at Birmingham (UAB).
